# Controls

## Summary
현재 조작 방향은 WASD 이동 + 마우스 방향/조준 기반이다.

## Current Decision

| 입력 | 역할 | 상태 |
|---|---|---|
| WASD | 이동 | 확정 |
| Mouse | 방향 / 조준 | 확정 |
| Left Click | 기본 공격 또는 현재 무기 액션 | 후보 |
| Right Click | 조준 / 차징 / 보조 공격 | 후보 |
| Shift | 대시 또는 순간이동 | 후보 |
| Space | 점프 | 후보 |
| Q / E | 부적 또는 장비/오행 전환 | 후보 |
| 1 / 2 / 3 / 4 | 오행 계열 스킬 | 후보 |
| Tab | 미정 | 보류 |

## Principles

- 조작은 액션과 탐험에 맞춰 설계한다.
- 클릭 이동은 현재 방향과 맞지 않으므로 사용하지 않는다.
- 플레이어가 자주 쓰는 입력부터 먼저 확정한다.
- 오행과 무기는 연결될 수 있다.
- 오행은 단순 속성이 아니라 플레이 방식에도 영향을 주는 방향으로 본다.
- 적에 따라 공략법이 달라질 수 있다.
- 하나의 전투에 하나의 정답만 강제하지 않는다.

## Open Questions

- 좌클릭은 항상 동일한 기본 공격인가, 장비/무기에 따라 달라지는가?
- 우클릭은 조준인가, 차징인가, 보조 공격인가?
- Q/E는 부적 전환인가, 무기 전환인가, 오행 전환인가?
- Tab은 무기 휠인가, 인벤토리인가, 비워둘 것인가?
- Shift는 대시인가, 순간이동인가?
- Space 점프는 전투용인가, 탐험용인가?
